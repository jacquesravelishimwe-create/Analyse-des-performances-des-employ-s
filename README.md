# Analyse-des-performances-des-employé-s

## Description
Dans ce projet, on effectue une analyse statistique pour comparer les groupes et proposer des solutions concrètes pour l'augmentation des performances.

# Objectif
1. Identifier les besoins en formations
2. Comparer les performances entre les différents groupes d'employés
3. Formuler des recommandations

# Etapes réalisées
1. Analyse de forme
2. Analyse de fond
3. Analyse plus détaillée
4. Preprocessing
5. Comparaison des performances des différents groupes 

# Technologies utilisées
1. Python
2. Pandas
3. Scikit-learn
4. Jupyter Notebook
